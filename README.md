
# Banana Ripeness Classification

YOLOv8 Classification application for predicting banana ripeness.

## Features

- Upload image
- YOLOv8 inference
- Confidence visualization
- Model comparison table
- Streamlit interface

## Project Structure

- app.py
- best.pt
- requirements.txt

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
